# Fire Ants II

#### By Will Johnson

## Description

An early attempt during Intro to Programming to create a text based adventure RPG. At the time I got pretty quickly due to my skill at the time, but the character creator menu served as a prototype for the one in "'Fun' Fighters".

## Setup/Installation Requirements

* Clone project
* Open Index.html
